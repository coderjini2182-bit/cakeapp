# 🎂 생일 케이크 메이커 PWA - 설치 가이드

## 📁 파일 구성
```
birthday-cake-pwa/
├── index.html        ← 메인 앱
├── manifest.json     ← PWA 설정
├── sw.js             ← 서비스 워커 (오프라인 지원)
├── 설치가이드.md
└── icons/
    ├── icon-192x192.png
    └── icon-512x512.png
```

## 🚀 설치 방법

PWA는 **웹서버에서 호스팅**해야 설치가 됩니다.
아래 무료 방법 중 하나를 선택하세요!

---

### 방법 1: GitHub Pages (추천 - 무료, 가장 쉬움)

1. **GitHub 계정 만들기** → https://github.com
2. **새 저장소(Repository) 만들기** → "birthday-cake" 같은 이름
3. **파일 업로드** → zip 풀어서 모든 파일을 저장소에 업로드
4. **Settings → Pages** → Branch를 `main` 선택 → Save
5. 1-2분 후 `https://내아이디.github.io/birthday-cake/` 주소 생성!
6. **휴대폰으로 이 주소 접속** → 앱처럼 설치 가능

---

### 방법 2: Netlify Drop (가장 빠름 - 1분)

1. https://app.netlify.com/drop 접속
2. zip 파일을 풀어낸 **폴더 전체**를 드래그 앤 드롭
3. 바로 URL이 생성됨!
4. 휴대폰으로 해당 URL 접속

---

### 방법 3: PC에서 로컬 테스트

터미널/명령 프롬프트에서:
```bash
cd birthday-cake-pwa
python3 -m http.server 8080
```
→ 같은 와이파이의 휴대폰에서 `http://PC아이피:8080` 접속

---

## 📱 모바일에서 앱으로 설치하기

### iPhone (Safari)
1. 위 URL을 **Safari**로 접속
2. 하단 **공유 버튼** (□↑) 탭
3. **"홈 화면에 추가"** 선택
4. 앱 이름 확인 후 **"추가"**
5. 홈 화면에 앱 아이콘 생성 완료!

### Android (Chrome)
1. 위 URL을 **Chrome**으로 접속
2. 상단 주소창 또는 하단에 **"앱 설치"** 배너 표시
3. 또는 **⋮ 메뉴 → "앱 설치"** 또는 **"홈 화면에 추가"**
4. **"설치"** 탭
5. 홈 화면에 앱 아이콘 생성 완료!
